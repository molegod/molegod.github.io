Rewind

HackMIT 2024 - Intersystems AI Challenge 3rd Place

A memory preservation web app for Apple Vision Pro

Key Features

  • Store and revisit memories in multiple formats (video, photos, text, audio)
  • Natural language search for memory retrieval
  • 3D scene generation using Gaussian splatting
  • WebXR integration for immersive memory viewing
View Project

Project Details

Memory Storage

Developed a comprehensive system for storing memories in various formats, allowing users to preserve their experiences in the most meaningful way.

3D Scene Generation

Implemented Gaussian splatting to generate immersive 3D scenes from user-uploaded videos, creating a more engaging way to revisit memories.

WebXR Integration

Created a WebXR interface using Three.js that allows Apple Vision Pro users to explore their memories in an immersive 3D environment.
