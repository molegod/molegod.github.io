NeuroScent

MIT Reality Hack 2025 - Hardware: Smart Sensing Winner

A VR brain-computer interface system with olfactory display for mental well-being

Key Features

  • OpenBCI's Galea BCI VR Headset integration with Unity
  • Dynamic interactive environment based on alpha brain waves
  • Custom-built olfactory display (~$60) with dual scent capability
  • ESP32 micro-controller firmware for USB serial connection
View on Devpost

Project Details

Brain-Computer Interface

Utilized OpenBCI's Galea BCI VR Headset to detect and respond to alpha brain waves, creating an immersive experience that adapts to the user's mental state.

Olfactory Display

Built a cost-effective olfactory display system using rapid-prototyping components and two diffusers, capable of delivering lavender and orange scents.

Unity Integration

Developed custom Unity scripts to create a dynamic VR environment that responds to brain wave patterns, enhancing the user's mental well-being.
