Thor Hammer

High School Senior Project

An electromagnetic Thor hammer with fingerprint authentication

Key Features

  • Dual heavy-duty door electromagnets
  • Arduino-based control system
  • Fingerprint scanner authentication
  • 48V power system (4x 12V lead-acid batteries)
Credit: Allen Pan on Youtube

Project Details

Electromagnetic System

Implemented a powerful electromagnetic system using two heavy-duty door electromagnets for secure hammer attachment.

Authentication System

Integrated a fingerprint scanner with Arduino for secure access control to the hammer.

Power System

Designed a robust power system using four 12V lead-acid batteries wired in parallel for reliable operation.
