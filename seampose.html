SeamPose PCB

Research Project - UIST 2024

A printed circuit board for upper-body pose estimation through capacitive sensing

Key Features

  • 8 conductive seams for capacitive sensing
  • Real-time upper-body pose estimation
  • Integration with vision-based ground truth
  • Machine learning model for pose prediction
View Paper

Project Details

Capacitive Sensing

Designed a PCB with 8 conductive seams that enable precise capacitive sensing for upper-body pose tracking.

Data Collection

Combined capacitive signals with vision-based ground truth to create a comprehensive dataset for training.

Machine Learning

Developed a machine learning model that accurately predicts upper-body pose from capacitive sensor data.
